## Sales Revenue Dashboard – Power BI
## 📌 Project Overview

This project showcases an interactive Sales Revenue Dashboard built using Microsoft Power BI. The dashboard provides insights into business performance by analyzing revenue, transactions, sales representatives, regions, stores, and product performance.
The goal of this dashboard is to help businesses monitor key metrics, identify trends, and make data-driven decisions.
